# 2048 Full-Stack Project 


## Project Overview
The full-stack 2048 game demonstrates:
- Frontend and backend integration
- User authentication
- Game state management and UI animations
- Client-server communication

### Game features
- Log in, register, or play as guest
- 4x4 game board with animated tile movements 
- Restart, Save, and Load game functionality
- Leaderboard displaying top scores

### Backend
- Uses SQLite with 3 tables: `users`, `scores`, and `saved_games`
- Passwords are encrypted with bcrypt
- Tracks user scores and saved game boards
- Provides API endpoints for login, registration, scoring, and game save/load

### Frontend
- Built with HTML, CSS, and JavaScripti02
- Communicates with the backend via fetch requests 
- Responsive 4x4 game board with interactive controls
